# simple-calculator
A simple calculator built using HTML, CSS, and JavaScript. Supports basic arithmetic operations.

Security Note:
In this project, I used eval() for simplicity. However, I am aware that eval() can pose security risks in production environments due to code injection vulnerabilities.
In this specific project, it was used because the application is a static, client-side tool with no backend or sensitive user data, making it a safe environment for demonstrating basic arithmetic logic.
